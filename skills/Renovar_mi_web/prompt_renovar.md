# 🚀 SUPERPROMPT: LANDING MIGRATION ENGINE (Transformación de Web Existente a Landing Optimizada)

## 🎯 OBJETIVO

Transformar una web ya existente (a partir de una URL y/o materiales aportados) en una **landing moderna, clara y optimizada para conversión**, respetando el branding y reutilizando assets cuando sea posible.

---

## 🧭 MICROAJUSTES ACTIVADOS (NUEVOS)

### A) MODO DE MIGRACIÓN (OBLIGATORIO ELEGIR UNO)

Antes de empezar, el usuario debe elegir el modo:

- **Conservadora**: Mantener estructura y tono en gran medida. Cambios mínimos necesarios para mejorar claridad/UX/performance.
- **Equilibrada (por defecto)**: Mantener esencia y secciones clave, pero reordenar/reformular para mejorar conversión.
- **Agresiva**: Reestructuración fuerte orientada a conversión. Simplifica, elimina ruido, reescribe copy y rediseña secciones.

> Regla: el modo elegido controla cuánto puedes cambiar copy/estructura/visual. No te salgas del modo sin permiso.

### B) CRITERIO DE ENTREGA (LANDING ÚNICA VS MULTIPÁGINA)

Antes de generar el proyecto, el usuario debe elegir:

- **One-page Landing (por defecto)**: Todo en una sola página (index.html) con anclas y scroll.

### 4) Placeholders (control estricto)
